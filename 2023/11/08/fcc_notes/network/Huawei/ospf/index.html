华为OSPF配置

进入OSPF并配置RID

ospf <1-65535> router-id <X.X.X.X>

这样就可以进入一个指定的ospf进程并手动指定rid

进入指定区域

area <0-4294967295> OSPF area ID (Integer)

area <X.X.X.X> OSPF area ID (IP address)

两种不同方式进入一个ospf区域(整数型更常用)

area 0是骨干区域，其他区域都必须与此区域相连。

宣告网段

network <自己的直连网络> <通配符掩码>

通配符掩码又称反掩码, 使用正掩码取反得到的

在启动ospf服务的每一台路由都执行一遍宣告网段的操作, 他们就会互相学习, 补充自己的LSA并合成LSDB

配置开销

如果有冗余路由的话, 我们自然不会让他形成环路

可以使用更改逻辑开销的方式使其编程备用链路

  • 进入需要修改的端口

interface <interfacetype> <interfacenum>

  • 修改cost

ospf cost <1-65535>
